## Запуск приложения
1. Клонируйте репозиторий
 ```bash
git clone https://github.com/EgorTolbaev/weatherApp.git
```
2. Установите зависимости
 ```bash
npm install
```
3. Запустите проект
-   На эмуляторе iOS
 ```bash
npm run ios
```
-   На эмуляторе Android
 ```bash
npm run android
```
-   На своем устройстве используя QR-код и Expo
 ```bash
npm start
```
